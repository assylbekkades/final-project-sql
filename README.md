# Final Project - SQL Analysis

## Описание
Анализ данных клиентов и транзакций розничного магазина
за период с 01.06.2015 по 01.06.2016.

## Данные
- `customer_info.xlsx` — информация о клиентах (2429 клиентов)
- `transactions_info.xlsx` — транзакции за год (419 122 записи)

## Задания
1. Список клиентов с непрерывной историей за год
   - средний чек за период
   - средняя сумма покупок за месяц
   - количество операций за период

2. Анализ в разрезе месяцев
   - средняя сумма чека в месяц
   - среднее количество операций
   - среднее количество активных клиентов
   - доля операций и суммы от годовых значений
   - % соотношение M/F/NA с долей затрат

3. Возрастные группы клиентов
   - шаг 10 лет + отдельно клиенты без возраста
   - сумма и количество операций за весь период
   - поквартальные средние показатели и %

## Инструменты
- MySQL 8.0
- MySQL Workbench

## Структура файлов
- `queries.sql` — все SQL запросы с комментариями
